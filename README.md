# simple-admin

> 基于vue的简易后台管理系统

## Build Setup

```bash
git clone https://github.com/xxzhiwei/simple-admin.git

cd simple-admin

npm install

npm run dev
```

## Build

```bash
npm run build:stage

npm run build:prod
```

## Advanced

```bash
npm run preview

npm run preview -- --report

npm run lint

npm run lint -- --fix
```
